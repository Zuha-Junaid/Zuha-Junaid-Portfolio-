Portfolio

1. Sports Academy Management System (DBMS)

**Objective:** Streamline the operations of a multi-sport training academy. This system manages athlete registrations, coach schedules, sports facilities, session bookings, and performance tracking. It provides a vital communication platform for athletes, coaches, and management, ensuring smooth operation and a better training environment.

2. Diabetes Prediction Model (Machine Learning)

**Objective:** Address the crucial problem of early diabetes prediction for timely treatment. I built an intelligent system using machine learning models (KNN, Decision Tree, Random Forest) to automate diagnosis based on key indicators like Glucose, BMI, Age, Insulin, and Blood Pressure, ultimately helping doctors and patients make informed decisions.

3. Al-Insaan Care (Mobile Application Design)

**Objective:** Design a mobile application to support illiterate and needy individuals by eliminating text-based barriers. The app provides financial or material aid using a **visual and audio-centric interface**. Users submit requests via voice messages/calls, which are validated by administration, ensuring inclusive and effective support delivery.

4. Information Security Project

The primary objective of this project was to identify and mitigate security vulnerabilities within a custom-built academic portal. By implementing a defense-in-depth strategy, the assessment utilized multiple security testing methodologies to ensure a robust security posture. The evaluation revealed moderate security risks, primarily involving server misconfigurations and missing security headers.

5. Research Nexus Website

Research Nexus is a modern, single-page academic collaboration platform built with Laravel 12, React, and Inertia.js. It allows researchers to upload manuscripts, share abstracts, and manage academic papers in a sleek, dark-mode-ready interface.

6. Operating Systems Project

Readers & Writers Problem with Writer's Priority This repository contains a Python implementation of the classical Readers-Writers synchronization problem, specifically designed with writer priority to prevent writer starvation. Developed as an Operating Systems semester project, it demonstrates the practical application of thread coordination, mutual exclusion, and resource management.

7. Emotion Recognition System

A Deep Learning project that classifies human facial expressions into 7 categories using a Custom CNN and a Streamlit dashboard.

8. Academic App

This is a one-page academic management system developed using Flask (Python) and MySQL (XAMPP). The system supports Admin, Student, and Teacher roles and performs full CRUD operations (Create, Read, Update, Delete) on users. The project is designed for academic/university submission and demonstrates backend development, database connectivity, and role-based routing.
